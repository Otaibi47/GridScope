What It Does

Core Capabilities

Topology Visualization
Live single-line diagram rendering showing substation layout, IED placement, bus bars, and breaker states with real-time color-coded status.
Analytics Dashboard
Trend charts, load curves, and frequency tracking using Matplotlib. Historical data aggregation with Pandas for pattern identification.
Event Replay
Scrub through timestamped event logs to reconstruct fault sequences. Replay any incident frame-by-frame for root-cause analysis.
Alarm Correlation
Groups related alarms into incidents automatically. Suppresses noise cascades and surfaces root alarms with severity ranking and timestamps.
Modern Desktop UI
Built with CustomTkinter/PyQt for a clean, responsive native interface. Resizable panels, dark theme, and operator-focused layout.
IEC 61850 Awareness
Designed around IEC 61850 concepts — logical nodes, datasets, and GOOSE messaging model — making it realistic for substation automation contexts.
